# 项目简介

基于深度学习的垃圾分类检测系统，采用YOLOv10算法与定制YOLO数据集，实现了对垃圾图片与视频流的实时分类与检测功能。系统包括UI界面、模型训练及推理模块，采用Python编写，便于操作与二次开发。技术特点在于高检测精度与速度，有效提升了垃圾分类处理的自动化水平。


## 核心功能
- 实时分类与检测垃圾图片与视频流
- 包含UI界面操作模块
- 提供模型训练功能
- 支持模型推理应用
- 便于操作与二次开发

## 适用场景
- 垃圾分类处理自动化场景
- 需要实时垃圾识别的环境
- 垃圾分类系统二次开发
- 垃圾分类模型训练与研究

## 技术特点
- 基于YOLOv10算法实现
- 采用定制YOLO数据集
- 使用Python语言编写
- 高检测精度与速度
- 有效提升垃圾分类处理自动化水平
